# 软件著作权申请辅助说明

## 软件名称

面向大语言模型的模态逻辑推理数据集构建与评测系统 V1.0

## 主要模块

1. 用户系统：注册、登录、退出、修改密码、管理员管理。
2. 题目生成：按模态类型、逻辑系统、难度、嵌套深度、题目格式和数量生成题目草稿。
3. 题目管理：手动录入、批量导入、筛选、编辑、确认、删除。
4. 数据集管理：创建数据集、加入已确认题目、查看统计分布、导出JSON。
5. 模板管理：维护 Zero-shot、CoT、Few-shot 等提示词模板。
6. 批量评测：多模型、多策略组合评测，记录原始输出、解析答案、正确性和失败信息。
7. 报告导出：按总体、模态类型、逻辑系统、难度、题目格式、模型和策略统计准确率。

## 源代码说明

- `backend/app/main.py`：系统接口、业务流程、评测任务和统计报告。
- `backend/app/auth.py`：用户密码哈希、登录令牌、权限校验。
- `backend/app/llm.py`：OpenAI API 格式调用、题目生成、答案解析。
- `backend/app/database.py`：SQLite连接和初始化。
- `database/init.sql`：数据库表结构。
- `frontend/src/App.vue`：中文前端页面和交互逻辑。
- `frontend/src/api.js`：前端请求封装和导出下载。
- `frontend/src/styles.css`：页面布局和视觉样式。

## 数据保护设计

- 普通用户只能访问自己创建的题目、数据集和评测记录。
- 管理员可查看全量用户数据，并具有后台管理入口。
- 密码使用 PBKDF2 哈希保存，不保存明文密码。
- 模型 `api_key` 只参与本次评测运行，不写入 SQLite 数据库。
